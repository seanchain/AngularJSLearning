directive()这个方法定义指令，指令可以理解为在特定DOM元素上运行的函数

directive()接受两个参数，分别为name和factory_function写法如下


angular.application('MyApp', [])
.directive('myDirective', function() {
            return {
                //通过设置项来定义指令，在这里可以进行覆写.
            }
        })
